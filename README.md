# Calendar-Component
用vue实现一款属于自己的日历组件

#### 功能实现
1. 在最上面展示的是我们当前的日期
2. 在下面也会给我们加一个另类的样式凸显出来我们当前的日期
3. 还可以去进行点击其他日期 再点击的时候最上面是展示我们点击哪一个就展示哪一个
4. 还可以进行月份的切换

#### 用到的知识
-  模板字符串
   - 和普通字符串的使用一样 
   - 多行字符串 
   - 字符串中嵌入变量 
   - 模板字符串中还可以调用函数 
   - 模板字符串之间还可以进行嵌套
-  动态class
-  事件绑定
-  computed
